Portfolio Web

-

30/03/2022 - 26/03/2023

Desarrollo de una aplicación web full stack, que muestra datos personales, estudios cursados, experiencia laboral y conocimiento de las tecnologías. Esta aplicación es de arquitectura distribuida y contiene un diseño de interfaz de usuario (Front end) que muestra la información, una base de datos que almacene los datos y debe contiene con las APIs necesarias para proveer a través de internet la información (Back end).
